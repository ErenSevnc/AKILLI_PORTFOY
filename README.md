AkilliPortfoy AkilliPortfoy, BIST hisseleri, kripto paralar, döviz ve emtiaları gerçek zamanlı olarak takip etmek için geliştirilmiş, teknik analiz ve backtest yeteneklerine sahip tam yığın bir portföy yönetim uygulamasıdır. Modern teknoloji yığını ve katmanlı mimari anlayışı ile sağlam bir temel üzerine kurulmuştur.🚀 ÖzelliklerCanlı Takip: BIST, Kripto, Döviz ve Emtia için canlı piyasa özeti ve kategoriye göre filtrelenmiş güncel haberler.Gelişmiş Portföy Yönetimi: Portföy CRUD işlemleri, Pie chart dağılımı, Kâr/Zarar hesapları ve yüzde değişimleri.Teknik Analiz: Recharts ile hazırlanmış interaktif grafikler üzerinden RSI ve Hareketli Ortalama (MA) indikatörleri.Optimizasyon ve Backtest: Özelleştirilebilir dönemlere sahip backtest sistemi ve tüm periyotlar için optimizasyon doğrulaması yapan MA/RSI motoru.Kapsamlı Veri: 132'den fazla desteklenen sembol, sembol autocomplete araması ve TRY, USD, EUR arasında anında para birimi dönüşümü.Güvenlik: PBKDF2-HMAC ve Custom JWT ile güvenli kullanıcı kimlik doğrulaması.🛠 Teknoloji YığınıFrontend:Framework: React (v19.2.0) Build Tool: Vite Routing: React Router UI & Grafikler: Recharts ve Lucide React HTTP İstemcisi: Axios Backend:Framework: FastAPI ve Uvicorn ASGI Server Finansal Veri: yfinance Veri İşleme: pandas Veritabanı: SQLite 🗺️ Geliştirme Yol HaritasıBu proje aktif geliştirme (Prototype) aşamasındadır. Gelecek planları şunları içerir:Kısa Vade: Environment variable (API URL ve gizli anahtarlar) yönetimi, React error boundary/loading skeleton eklemeleri ve eksik Optimizasyon sayfalarının tamamlanması.Orta Vade: Frontend tarafında TypeScript'e geçiş, büyük componentlerin parçalanması ve Vitest/Jest ile test coverage sağlanması.Uzun Vade: Üretim ortamı veritabanı için PostgreSQL'e geçiş, sık çekilen veriler için Redis Cache katmanı ve gerçek zamanlı anlık fiyat akışı için WebSocket entegrasyonu.
